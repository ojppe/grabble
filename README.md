# [Grabble](https://grabble.org)
Open Scrabble server based on [lila](https://github.com)

Still in early development. features will include:
- Tournaments
- Multiplayer 2-4
- Customizable board layout
- Different time controls
- Definition lookup
- Game archive
- Statistics
- Rankings
- Analysis
- Puzzles
- Trainer
